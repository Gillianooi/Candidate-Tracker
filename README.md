# Candidate Tracker – ASEAN / Asia Records

Mobile-first candidate follow-up tracker with AI-powered updates and WhatsApp draft generation.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Login with GitHub
3. Click "Add New Project" → Select this repo
4. Framework: **Vite** (auto-detected)
5. Click **Deploy**

Done! You'll get a live link like `your-app.vercel.app`

## Local Development

```bash
npm install
npm run dev
```
